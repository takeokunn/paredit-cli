# Code of Conduct

## Our standards

Contributors and maintainers are expected to act professionally and
constructively. In particular:

- Be respectful of differing experience levels, backgrounds, and viewpoints.
- Give technical feedback about code, behavior, and evidence rather than people.
- Welcome good-faith questions and help others participate effectively.
- Avoid harassment, discrimination, personal attacks, and sustained disruption.

## Scope

This code of conduct applies to project spaces, including issues, pull
requests, documentation discussions, and any direct communication that
represents the project.

## Enforcement

Maintainers may remove content, close discussions, or limit participation when
behavior violates these standards. Report concerns privately through the
repository owner's [GitHub profile](https://github.com/takeokunn), rather than
escalating them in a public thread.
